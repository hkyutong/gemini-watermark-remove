# Gemini 无损去水印工具（中文文档）

主文档请查看 [README.md](./README.md)。

## 快速开始

```bash
corepack pnpm install
corepack pnpm dev
npx --yes serve dist -l 28008
```

访问：`http://127.0.0.1:28008`

## 生产构建

```bash
corepack pnpm build
```

## 用户脚本产物

# YutoAI 无损去水印工具（中文文档）

主文档请查看 [README.md](./README.md)。

## 本地运行

```bash
pnpm install
pnpm dev
pnpm serve
```

生产构建：

```bash
pnpm build
```

用户脚本产物：

- `dist/userscript/yutoai-watermark-remover.user.js`
